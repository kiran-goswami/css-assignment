what is css?

  1. CSS stands for Cascading style sheet

  2. CSS describes how HTML elements are to be displayed on screen,paper,or in other media

  3. Css is not neccesary CSS saves a lot of work.It can control the layout of multiple web pages all at once

  4. external stylesheet are stored in css files

CSS can be added to html document in 3 ways:

  • Inline - by using the style attribute inside HTML elements 1
  • Internal - by using style element in the head section
  • External - by using a link element to link an external CSS file .
  • The most comman way to add CSS , is to keep the styles in external CSS files . However , in this tutorial we will use inline and internal styles , because this is easier to demonstrate , and easier for you to try it youurself .

When comparing to CSS class vs ID, the difference is that CSS class applies a style to miltiple elements.ID , on the other hand applies a style to one unique element. ID is also special in that you can use a special URL to link directly and it's used by javascript.

In CSS, selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted , a style can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
